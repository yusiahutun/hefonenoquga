What is a bank code?

A "bank code" is a series of number that helps identify banks across the world. Each bank has its own specific and unique code that differentiate it from all those on its country as well as all others in the world. This means that all banks in Kenya have different bank branch codes.

The Equity Bank code is 68. Note that the number should be written as "68" and not "068". Whenever you are asked to input your bank code details the number will be up-to 5 digits. While Equity Bank's code is 68, the other 3 digits are known as "branch codes" and these 3 digits will specify exactly which branch of the larger Equity Bank that your account belongs to.

What is a swift code in a bank account?

A swift code is a standard format of bank identifier codes and is a unique identifier code for a bank. These codes are mostly used in international transactions especially when transferring money using wire transfers. Banks also use these swift codes when communicating with each other. Unlike codes that have 5 characters, bank swift codes have about 8 to 11 characters and also have a different format.

SWIFT CODE FORMAT - AAAA BB CC DDD

AAA - bank code (only letters)

BB - country code (only letters) KEN

CC - location code (letters and digits)

DDD - branch code, this is optional.

The Equity bank Kenya swift code is EQBLKENA. Remember that a SWIFT transfer takes 2-3 days to reflect in your account.
